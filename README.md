# Coffee Shop Website

A modern, responsive coffee shop website built using **HTML, CSS, and JavaScript**.  
This project showcases a complete front-end layout for a caf√© business, including menu sections, testimonials, gallery, and contact information.

---

##  Features

- Responsive design (mobile, tablet, desktop)
- Sticky navigation bar with mobile menu
- Hero section with call-to-action buttons
- About Us section with social media links
- Menu showcase
- Testimonials slider using **Swiper.js**
- Image gallery with hover effects
- Contact section with form layout
- Clean and modern UI design

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Swiper.js** (for testimonials slider)
- **Font Awesome** (icons)
- **Google Fonts**
